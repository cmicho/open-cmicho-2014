PNI : Adobe Phonegap Build
Objectif : créer et partager des applications prêtes à l'emploi pour les principaux terminaux mobiles (iOS, Android, Windows Phone, Blackberry, webOS et Symbian) avec le service Adobe Phonegap Build
Prérequis : connaitre les rudiments de l'HTML et des CSS
Versions : 2.3.0
Durée minimum : 1 jour

L’apprentissage se fait avec la création de bout en bout d’une publication. Il s’agit ici d’assimiler toutes les étapes du processus de création plutôt que d’apprendre la création elle même. La maîtrise des codes HTML5, CSS3 et JavaScript pour la création d’un site web mobile (base d’une app phonegap) est à appréhender dans le cadre d’une autre formation, nécessairement plus longue.
